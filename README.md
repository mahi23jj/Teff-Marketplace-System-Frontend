# 🌾 TeffMarket Frontend

A modern, role-based marketplace platform that connects **farmers** directly with **buyers**, making it easier to trade high-quality teff while promoting transparency, fair pricing, and efficient order management.

The platform empowers farmers to showcase their products, monitor market trends, and manage sales, while giving customers a seamless shopping experience and administrators the tools needed to oversee the marketplace.

---

## ✨ Features

### 👨‍🌾 Farmer Dashboard

* Manage and publish teff products
* Update product inventory and pricing
* Track incoming and completed orders
* View real-time market price trends with interactive charts
* Monitor sales performance
* Manage profile and farm information

### 🛒 Customer Dashboard

* Browse teff products from different farmers
* Search and filter products
* View detailed product information
* Place and manage orders
* Track order status
* Manage personal profile

### 🛠️ Admin Dashboard

* Manage farmers and customer accounts
* Approve and monitor marketplace activities
* View platform analytics
* Monitor market price trends
* Manage products and orders
* Maintain overall marketplace integrity

## ⚙️ Getting Started

### Prerequisites

* Node.js 18+
* npm or yarn

### Installation

```bash
git clone https://github.com/your-username/teffmarket-frontend.git

cd teffmarket-frontend

npm install

npm run dev
```

The application will be available at:

```
https://teff-marketplace-system-frontend.vercel.app
```
